## 🌐 Overview

In a world where data breaches and unauthorized access are becoming increasingly common, protecting sensitive files is more important than ever.

This project is a **Python-based File Encryption & Decryption Tool** that enables users to securely encrypt and decrypt files using **AES-256**, one of the most trusted encryption standards used by governments, financial institutions, and cybersecurity professionals worldwide.

Whether it's documents, PDFs, images, source code, or confidential reports, this tool ensures that your data remains unreadable to anyone without the correct password.

---

## 🎯 Key Features

🔒 **AES-256 Encryption** for strong data protection

🛡️ **AES-GCM Mode** providing both confidentiality and integrity

🔑 **Password-Based Security** using PBKDF2 key derivation

📂 **Supports Any File Type**
- Documents
- PDFs
- Images
- Videos
- Source Code
- Archives

🖥️ **User-Friendly GUI** built with Tkinter

⚡ Fast Encryption & Decryption Process

🚫 Protection Against Unauthorized Access

## ⚙️ How It Works

```text
Plain File
    │
    ▼
🔑 User Password
    │
    ▼
PBKDF2 Key Derivation
    │
    ▼
🛡️ AES-256 Encryption
    │
    ▼
Encrypted File (.enc)
```

For decryption:

```text
Encrypted File
      │
      ▼
Correct Password
      │
      ▼
AES-256 Decryption
      │
      ▼
Original File Restored
```
